# 远程访问

远程访问页面用于向支持弹弹 Play 远程访问协议的客户端提供连接信息。系统会根据当前站点配置决定是否允许访问，以及是否需要令牌授权。

## 前置条件

使用远程访问前，请先在后台“站点配置 > 远程访问”中确认：

- 已开启远程访问
- 如启用了令牌授权，已设置所需角色
- 当前账号具备访问要求的角色

## 页面内容说明

页面主要分为三部分：

- 二维码区域：供客户端扫码导入连接信息。
- 连接信息区域：显示主机地址、端口、当前用户、是否需要授权、所需角色。
- 原始数据区域：展示二维码对应的 JSON 内容，并支持复制。

## 扫码连接

![image-20260321191642271](/remote-access/image-20260321191642271.jpg)

1. 打开[弹弹play官方的安卓app](https://www.dandanplay.com/)，选择【PC端媒体库】
2. 使用扫码快捷录入功能扫描页面中的二维码。
3. 如果你开启了权限限制，可能需要手动填写token

建议：

- 优先在同一局域网内测试。
- 如果浏览器地址是内网 IP，客户端也应能访问该 IP。
- 如果页面通过域名访问，确认客户端网络能够解析并访问该域名。

## 常见问题

### 页面提示远程访问已关闭

请到后台“站点配置 > 远程访问”中开启对应功能。

### 能打开页面但客户端连接失败

- 检查主机地址是否可达。
- 检查站点端口是否正确开放。
- 确认客户端与服务端是否在同一网络或可互通网络中。

### 扫码后仍提示无权限

- 检查当前用户角色是否满足远程访问要求。
- 如果开启了令牌授权，确认二维码中的令牌是最新生成的。
