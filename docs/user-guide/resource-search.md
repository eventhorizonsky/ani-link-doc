# 资源搜索

资源搜索用于从已配置的资源节点中检索番剧资源，并将结果直接创建为下载任务。

## 前置条件

开始使用前，建议先确认：

- 已在“资源接口与下载器配置”中填写资源节点地址，这个节点可以参考[弹弹官方文档](https://doc.dandanplay.com/faq/resource-service.html)

  ![image-20260321193208543](/resource-search/image-20260321193208543.jpg)

- 已存在至少一个可用媒体库

- 下载临时目录和目标媒体库路径可正常读写

## 页面说明

页面主要包含三部分：

- 搜索条件区：关键字、字幕组、资源类型、目标媒体库
- 搜索结果区：展示标题、类型、字幕组、大小、发布时间
- 下载任务区：展示任务状态和实时进度

![image-20260321193040772](/resource-search/image-20260321193040772.jpg)

## 搜索资源

1. 输入搜索关键字，通常建议使用番剧名。
2. 可按需选择字幕组。
3. 可按需选择资源类型。
4. 选择下载完成后要入库的目标媒体库。
5. 点击“搜索”。

如果结果过多，页面会提示“当前仅显示部分结果”，这通常意味着需要进一步缩小关键词范围。

## 创建下载任务

在搜索结果列表中，点击某条资源后的“下载”按钮即可创建任务。

提交时前端会附带以下信息：

- 标题
- 磁力链接
- 发布页地址
- 文件大小
- 发布时间
- 字幕组
- 资源类型
- 目标媒体库 ID

创建成功后，可在页面下方或“下载管理”页面查看进度。

![image-20260321193104517](/resource-search/image-20260321193104517.jpg)

## 实时进度

页面会通过 SSE 实时接收任务更新，因此你通常无需频繁手动刷新。

任务区常见能力包括：

- 刷新任务
- 取消任务
- 重试任务
- 删除任务
- 查看绑定状态

如果右上角或状态区域提示连接异常，说明实时进度流可能暂时断开，此时可以先手动刷新。

## 查看绑定状态

下载任务完成后，系统会尝试把最终文件与媒体库记录关联起来。

绑定状态中通常能看到：

- 任务状态
- 最终路径
- 媒体文件 ID
- 是否已进入媒体库
- 动画 ID / 标题
- 剧集 ID / 标题
- 匹配状态

这部分是排查“下载成功但库中没有显示”的关键入口。

![image-20260321193118280](/resource-search/image-20260321193118280.jpg)

## 使用建议

- 搜索前先选好目标媒体库，避免任务创建失败。
- 关键字过宽时，优先加上季度、年份或字幕组缩小范围。
- 下载后若未正确入库，可继续到“视频资源管理”或“媒体库管理”中核查。

## 常见问题

### 点击下载没有反应

- 检查是否已选择目标媒体库。
- 检查资源节点地址是否配置正确。
- 查看页面提示信息和浏览器控制台报错。

### 搜不到资源

- 确认资源节点可访问。
- 更换更精确或更宽松的关键字再试。
- 检查是否代理配置不正确。
