#### 📚 媒体库管理
- 添加和管理本地媒体库
- 自动扫描视频文件
- 实时文件变更监听
- 支持多媒体库配置

- 集成弹弹play番剧数据库

- HTML5视频播放
- HTTP Range支持（可拖拽进度）

- 多种音轨和字幕支持

#### 💬 弹幕功能
- 实时弹幕显示和缓存
- 弹幕信息搜索和过滤
- 支持多种弹幕格式
- 自定义弹幕显示参数

#### 📺 动漫库
- 完整动漫库浏览
- 全文搜索功能
- 动漫详细信息展示
- 周更新番时间表

#### ⚙️ 系统管理
- 系统信息查看
- 站点配置
- Dandan App配置
- 视频文件管理

## 常见场景

### 场景1：我有一个包含番剧视频的文件夹

1. 启动AniLinkServer
2. 进入**媒体库管理**页面
3. 点击**添加媒体库**，选择你的文件夹
4. 等待**自动扫描**完成
5. 访问**动漫库**查看已匹配的番剧

详见 [媒体库管理](./media-library.md)

### 场景2：我想搜索特定番剧

1. 进入**动漫库**页面
2. 使用顶部搜索框搜索番剧名称
3. 点击番剧卡片查看详情和本地剧集
4. 选择剧集开始播放

详见 [动漫浏览与搜索](./browsing.md)

### 场景3：我要修改某部番剧的信息

1. 进入**管理后台 -> 动漫库**
2. 搜索目标番剧
3. 点击**编辑**修改名称、封面等信息
4. 保存修改

详见 [系统配置](./settings.md#修改番剧信息)

### 场景4：弹幕功能不工作

检查以下项：

1. 确认视频来源支持弹幕（通常需要匹配的episodeId）
2. 检查**系统设置**中的弹幕源配置
3. 尝试刷新页面
4. 查看浏览器控制台是否有错误信息

详见 [故障排查 - 弹幕问题](./troubleshooting.md#弹幕相关)

## 重要提示

### 媒体库路径

- **不应只读** - 系统需要写入权限以更新文件索引
- **建议独立** - 放在专门的媒体库文件夹，避免混合其他文件
- **性能考虑** - 使用本地磁盘而非网络存储以获得最佳性能

### 数据备份

- **定期备份** - 定期备份 `./anilink/config` 和数据库
- **文件夹** - 如使用H2，还需备份 `./anilink/data` 文件夹
- **配置保留** - 重装系统时保留 `config` 文件夹以恢复配置

### 性能优化

- **禁用未使用功能** - 在管理后台禁用不需要的功能
- **定期维护** - 定期清理历史数据和缓存
- **硬件升级** - 处理大型库（>10000个文件）时考虑升级硬件
